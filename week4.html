Week 4

In de vierde week heb ik een screenshot van mijn treinborden genomen en die in ChatGPT gezet met de vraag om dit om te zetten naar code voor Visual Studio Code. De eerste versie van de code vond ik niet ideaal, dus heb ik gevraagd om die te verbeteren en daarna de nieuwe code ingeplakt in mijn project. Toen ik de pagina in mijn browser opende, leek het ontwerp nog niet volledig op mijn originele borden, dus heb ik zelf nog wat dingen aangepast tot het visueel dichter in de buurt kwam, al is het nog steeds niet exact wat ik in gedachten had.
